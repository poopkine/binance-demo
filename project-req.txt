Сделать личный кабинет пользователя, состоящий из одной страницы, на которой блоками размещена информация, взятая из API Binance.

  а) Портфель пользователя
  
  б) История покупок/продаж пользователя (не ордера)

  в) График(и) успешности торговли пользователя / стоимости портфеля

Размер блоков, размещение их на странице, наличие определенных полей в таблицах и все остальное - вы делаете на свой вкус.
По итогу должен получиться кабинет, в котором можно более наглядно и удобно увидеть свой портфель, статистику своей торговли, сводку, график.
